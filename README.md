Procrastination Assistant is a humorous browser extension that detects when users open study-heavy websites such as LeetCode, Wikipedia, or GeeksforGeeks and encourages them to take a break. The extension aims to highlight the relatable struggle of student procrastination in a fun and interactive way.

It includes several features such as:

An Excuse Generator to quickly create funny excuses

A Fake Study Timer that simulates productivity

A Distraction Finder that suggests entertainment websites

A Productivity Warning Popup that appears when users open study websites

The project demonstrates how browser extensions can interact with websites using Manifest V3, content scripts, HTML, CSS, and JavaScript while delivering a light-hearted user experience.
